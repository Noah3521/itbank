사용자의 입력값을 서버에 전송하기 위한 요소. form과 연계하여 사용한다

태그 type 특징 예시
input text 일반적인 텍스트를 입력받는다
password 마스킹된 텍스트를 입력받는다
number 숫자를 입력받는다 2000
date 날짜를 입력받는다
range 마우스로 범위 사이의 숫자를 입력받는다
file 마우스로 첨부파일을 입력받는다
radio 여러 요소중 해당하는 값을 복수 선택하여 입력받는다"
checkbox 강아지 고양이 거북이
색상값을 입력받는다
select 여러 선택지 중 하나를 입력받는닫 (radio와 방식이 유사하다)
